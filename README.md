XYZ implementation in Java.

To run, start a mongo DB instance:

`docker-compose up`

Then start the project using Spring Boot:

`mvn spring-boot:run`

The client is accessible at <http://localhost:9834/c>

The AS is accessible at <http://localhost:9834/as>
